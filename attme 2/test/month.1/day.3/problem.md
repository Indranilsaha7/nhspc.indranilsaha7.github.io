ধরে নাও, NHSPC-এর একটি গেমে তুমি কিছু পয়েন্ট পেয়েছ এবং একটি ভুল করার জন্য কিছু পয়েন্ট কাটা গেছে। তোমার কাজ হলো নিচের instruction অনুযায়ী কোড লেখা:

`wins` নামের একটি variable তৈরি করো এবং এর value দাও `7`।

`points_per_win` নামের একটি variable তৈরি করো এবং এর value দাও `10`।

`penalty` নামের একটি variable তৈরি করো এবং এর value দাও `5`।

এবার `total_score` নামের একটি নতুন variable তৈরি করো। এর ভেতরে `math operator` ব্যবহার করে হিসাব করো: `wins` এবং `points_per_win` গুণ করে তা থেকে `penalty` বিয়োগ করবে।

সবশেষে, শুধু `total_score` প্রিন্ট করে আউটপুট দেখাও।

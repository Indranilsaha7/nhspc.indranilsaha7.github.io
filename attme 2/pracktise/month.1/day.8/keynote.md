# Day 8: String Indexing & Basics 

## The Lesson:
Python-এ একটি String হলো মূলত অনেকগুলো character বা অক্ষরের একটি শেকল। তুমি যখন একটি String তৈরি করো, Python সেই String-এর প্রতিটি অক্ষরকে একটি নির্দিষ্ট পজিশন বা সিরিয়াল নম্বর দিয়ে দেয়। এই সিরিয়াল নম্বরটিকে বলা হয় Index। String থেকে নির্দিষ্ট কোনো অক্ষর বের করে আনার এই পদ্ধতিকেই বলা হয় String Indexing। Indexing করার জন্য আমরা স্কয়ার ব্র্যাকেট `[]` ব্যবহার করি।
### সবচেয়ে গুরুত্বপূর্ণ বিষয়: Python-এ গণনা 1 থেকে শুরু হয় না, শুরু হয় 0 থেকে!

## Tips, Tricks, Rules (Sutros), and Formulas:
* **Rule 1 - Zero-Based Indexing:** যেকোনো String-এর প্রথম অক্ষরের Index সবসময় `0` হবে। যদি String-এ ৫টি অক্ষর থাকে, তবে তার শেষ ইনডেক্স হবে `4`।
* **Rule 2 - IndexError:** তুমি যদি এমন কোনো Index কল করো যা **String**-এ নেই (যেমন `word[10]`), তবে **Python** কোডটি ক্র্যাশ করবে এবং `IndexError` দেখাবে। `Competitive Programming`-এ এই ভুলটি অনেক বেশি হয়।
* **Trick 1 - The Ultimate Tail (-1):** String-টি কত বড় তা যদি তোমার জানা না-ও থাকে, কোনো সমস্যা নেই! সবসময় `[-1]` ব্যবহার করলে তুমি একদম শেষ অক্ষরটি পেয়ে যাবে। একইভাবে `[-2]` দিলে পাবে শেষের দিকের দ্বিতীয় অক্ষরটি।

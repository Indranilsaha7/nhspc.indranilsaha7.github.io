# Today i will know the Variables & Data Types :
## What I will know in this lesson:

### Python-এ Variable হলো একটি পাত্র বা বাক্সের মতো, যেখানে Data সেভ করে রাখা হয়।
#### Data Type মূলত ৪ প্রকার:

1. Integer (int): পূর্ণ সংখ্যা (যেমন: 14, 2027)।
2. Float (float): দশমিক সংখ্যা (যেমন: 95.5)।
3. String (str): যেকোনো Text, যা অবশ্যই " " এর ভেতরে থাকবে (যেমন: "Indranil")।
4. Boolean (bool): এটি শুধু True অথবা False হয়।

# Day 4: Taking Inputs:
## Step 2: Tips, Tricks, Rules (Sutros), and Formulas

* Rule 1 - The Default String Rule: এটি Python-এর সবচেয়ে গুরুত্বপূর্ণ রুলগুলোর একটি। input() function সবসময় যেকোনো কিছুকে String (str) হিসেবে গ্রহণ করে। ইউজার যদি 2027 টাইপ করে, Python সেটাকে Number হিসেবে না ধরে Text ("2027") হিসেবে ধরবে।

* Trick 1 - Prompt Text: input()-এর ব্র্যাকেটের ভেতরে কোটেশন মার্ক দিয়ে তুমি একটি মেসেজ বা Prompt দেখাতে পারো (যেমন: input("Enter your age: "))। এটি দেখে ইউজার বুঝতে পারে তাকে ঠিক কী ইনপুট দিতে হবে।
